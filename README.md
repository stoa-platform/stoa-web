# 🏛️ STOA Web

Landing page for [STOA Platform](https://gostoa.dev) — The Cilium of API Management.

## 🚀 Stack

- [Astro](https://astro.build) 4.x
- [Tailwind CSS](https://tailwindcss.com) 3.x
- Deployed on [Vercel](https://vercel.com)

## 🛠️ Development
```bash
npm install
npm run dev
```

## 📦 Build
```bash
npm run build
```
